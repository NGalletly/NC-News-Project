Nevilles News API

This Api fetches articles where you can read the information and comments.
You can add your own comment or upvote hot takes(or downvote trash garbage trolls if you like).

Available Endpoints

  • GET /api/topics

    Retrieve a list of all topics.

    Example request and response:

    Example request:

                
            GET https://nevilles-news.onrender.com/api/topics

    Example Response:

      
{
  "topics": [
    {
      "slug": "coding",
      "description": "Code is love, code is life"

}

  • POST api/articles/:article_id/comments

    Post comment for article_id with {"username": "abc", "body":"xyz"} Note : currently the username MUST exist.

    Example request and response:

    Example request:

                
             POST: https://nevilles-news.onrender.com/api/articles/1/comments
             JSON body:
             {
               "username": "weegembump",
               "body": "Posting a comment"
              }

    Example response:

      
{
	"comment": {
		"comment_id": 321,
		"article_id": 1,
		"body": "Posting a comment",
		"votes": 0,
		"author": "weegembump",
		"created_at": "2026-02-21T15:14:31.125Z"
	}
}

  • PATCH api/articles/:article_id

    Updates votes for article ID using { "inc_votes": 1 }

    Example request and response:

    Example request:

                
             PATCH: https://nevilles-news.onrender.com/api/articles/1
             JSON body:
             {
               "inc_votes": 1
              }

  • DELETE /api/comments/:comment_id /h3>

    Deletes comment from comment table using parametric endpoint, eg : "/api/comments/140

    Example request and response:

    Example request:

                
            DELETE https://nevilles-news.onrender.com/api/articles/1

    Example response:

      
    Status code: 204,
    No body returned for response
