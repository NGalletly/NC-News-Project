NC News API

This Api fetches articles where you can read the information and comments.
You can add your own comment or upvote hot takes(or downvote trash garbage trolls if you like).

Available Endpoints

  • GET /api/topics

    Retrieve a list of all topics.

    Example response:

      
{
  "topics": [
    {
      "slug": "coding",
      "description": "Code is love, code is life"
    },
    {
      "slug": "football",
      "description": "FOOTIE!"
    },
    {
      "slug": "cooking",
      "description": "Hey good looking, what you got cooking?"
    }
  ]
}

  • POST api/articles/:article_id/comments

    Post comment for article_id with {"username": "abc", "body":"xyz"}

    Example response:

      
{
	"comment": {
		"comment_id": 331,
		"article_id": 1,
		"body": "Testing the service layer logic.",
		"votes": 0,
		"author": "weegembump",
		"created_at": "2026-02-10T14:18:14.965Z"
	}
}
